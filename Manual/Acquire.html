ACQUIRE MENU

This menu is used to control the data acquisition from an installed multichannel analyzer board such as the one from 4pi, Inc. If you have the board, put the SpectralEngine plug-in furnished with the board into the plug-ins folder that is in the folder with DTSA. The one plug-in will handle all the boards installed in your computer.

This version of the program supports multiple boards, therefore, multiple spectra may be simultaneously collected.

Assign Display(s)

This allows the user to assign a display for each board in use. It is no longer necessary to acquire spectra only into the Work Display. The Hardware Setup allows the user to set up each board and the Assign Meters is used to select which (if more than one) acquisition will be displayed in the Main Control Window meter.

The Stop, Continue and Clear/Start items control the individual boards or all of the boards.

The Automation is not currently active.
